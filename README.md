# Poronga js

## Easy to use:

Just `npm install @ivangarcia/poronga` and then:

```
import { Prng } from 'poronga';

const Component = () => {
  return (
    <Prng formData=[{ type: "text", label: "Name", placeholder: "Full name", name: "name" }] />
  )
}
```
### And then just see the magic happen!!
